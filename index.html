Project Idea

Overview

A project that I was thinking a couple of time is an app that automate computer to do file system management. The app could automate moving files, uninstalling unnecessary installs, control junk files, and simplify the use of scripting for many people without the knowledge of coding. This app provides flexibility for Windows/Mac users (Linux future compatibility) to have a companion that will help organise system files and software’s manually but more importantly automatically. This app should be completely usable when offline to also increase its security and does not affect the computer performance. This app will be called Valet.

Motivation

My motivation on this making this app is to create a very reliable and easy way to control my windows computer file system. Things that make me frustrated using a computer is when downloading many things could clutter the download folder and unnecessary software’s that is installed to the computer causing the computer to lose performance. This app idea motivation is mostly personal motivation to improve my daily life using the computer. I think this kind of software is needed to be made and there must be people who has the same problem as I am.

Description

First on what the app should do in the user computer. This app should start off with installation that will setup the file directory or folder which the app needs to focus on organising. User will then select the types of file the app should manage and where to make a new folder.

The app should be completely workable even when it’s not connected to the Internet. There will be two version of this app one which is a free standalone application which doesn’t need internet connection and one paid subscription plus mode that includes extra features.

Free version

  • Manages folder autonomously
  • Clears unnecessary files or program
  • Uninstalling unwanted vanilla software
  • Custom scripting program

Plus version

  • Manages folder autonomously with specific algorith for different data types
  • Clears unnecessary files or program
  • Uninstalling unwanted vanilla software
  • Custom scripting program
  • Support mobile device
  • Security update based on the user need
  • Set timed events for the app to do specified jobs (Ex. opening another app)

The app should have a graphic interface where the user can see information on what the app have been doing and it is where users could change their settings. On top of that, there should be a section for user to create custom program that can be executed by the app to furthermore expand the usability of the app.

The app would be lightweight, so it won’t affect the computer performance and supports majority of the computers. Unlike anti-virus software’s this app should not be disturbing the user workflow and works seamlessly with the user preferences.

Since there will be a limit on how powerful the app should be. The app will have drawbacks on how many features it will have progressively. But there can be another option on expanding the feature of the app by using cloud-based service. This means new or experimental feature of the app could be fetch from a server and managed by the developer which can lower the need of storage requirement. This is unnecessary but it could make the quality of app be controlled much easier by the developer.

The app must be totally functional even when its on the free version and the plus version should just be just for paying an expert to do it. The pricing can’t be decided now but it be a subscription method and not a one-time purchase. This could lead to users to stick to the free version but if the user is not familiar with the coding or scripting then they would prefer to pay extra to gain expert knowledge.
